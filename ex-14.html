# The Border Box


  • Most elements have a border
  • The way to make these borders visible is to add a border to elements using CSS "border" property
  • The "border" property is actually a shorthand that combines the values of multiple properties into one declaration:
    • "border-width" determines the "thickness" of the border, and is usually set with absolute units (i.e. pixels)
    • "border-style" includes values like "solid", "dashed", "dotted"
    • "border-color" can be set with a named color, an rgb() value, or a hexadecimal
